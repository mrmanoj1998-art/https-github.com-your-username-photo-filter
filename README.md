# 📸 Photo Filter Web App

A simple and interactive photo filter web application built using HTML, CSS, and JavaScript.

---

## 🚀 Features
- Upload image from your device
- Apply grayscale filter
- Instant preview on screen
- Fast and lightweight

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## ▶️ How to Use
1. Open the website
2. Upload an image
3. Click on "Grayscale" button
4. See the result instantly
